# AURA Wellness Spa — SSENSE-inspired redesign

Minimal editorial redesign for the AURA Wellness Spa React/Vite site.

## Run

```bash
npm install
npm run dev
```

## Replace

Copy these into your project:

- `src/App.jsx`
- `src/App.css`
- `src/main.jsx`
- `index.html`
- `package.json`

This version includes Lenis smooth/inertia scrolling.
